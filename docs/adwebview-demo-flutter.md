# Demo For Flutter AdWebView

Flutter WebView interactive advertising demo project.

## Project Location

Demo source code is located in the `AdWebViewFlutterDemo/` directory of the repository.

- [GitHub Repository](https://github.com/youmi-obg/Documentation/tree/main/AdWebViewFlutterDemo)

## Project Structure

```
AdWebViewFlutterDemo/
└── code/                       # Source code directory
    ├── lib/
    ├── android/
    ├── ios/
    └── pubspec.yaml
```

## Quick Start

1. Clone the repository
2. Enter the `AdWebViewFlutterDemo/code/` directory
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the project

## Main Features

- Flutter WebView integration
- Platform-specific configuration
- Youmi Flutter Plugin usage examples
