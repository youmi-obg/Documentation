# Demo For Flutter AdWebView

Flutter WebView 互动广告示例项目。

## 项目位置

Demo 源代码位于仓库的 `AdWebViewFlutterDemo/` 目录下。

- [GitHub 仓库](https://github.com/youmi-obg/Documentation/tree/main/AdWebViewFlutterDemo)

## 项目结构

```
AdWebViewFlutterDemo/
└── code/                       # 源代码目录
    ├── lib/
    ├── android/
    ├── ios/
    └── pubspec.yaml
```

## 快速开始

1. 克隆仓库
2. 进入 `AdWebViewFlutterDemo/code/` 目录
3. 运行 `flutter pub get` 安装依赖
4. 运行 `flutter run` 启动项目

## 主要功能

- Flutter WebView 集成
- 平台特定配置
- Youmi Flutter Plugin 使用示例
