# Demo For Android AdWebView

Android WebView interactive advertising demo project.

## Project Location

Demo source code is located in the `AdWebViewDemo/` directory of the repository.

- [GitHub Repository](https://github.com/youmi-obg/Documentation/tree/main/AdWebViewDemo)

## Project Structure

```
AdWebViewDemo/
├── AdWebViewDemo.apk          # Compiled APK file
└── code/                       # Source code directory
    ├── app/
    │   └── src/main/
    │       ├── java/com/example/adwebviewdemo/
    │       │   ├── MainActivity.java
    │       │   ├── WebViewActivity.java
    │       │   └── utils/OpenAppUtils.java
    │       └── res/
    └── build.gradle
```

## Quick Start

1. Clone the repository
2. Open the `AdWebViewDemo/code/` directory with Android Studio
3. Sync Gradle dependencies
4. Run the project

## Main Features

- WebView basic configuration
- Download listener handling
- URL redirection handling
- JavaScript interface bridging
- Web page back navigation support
